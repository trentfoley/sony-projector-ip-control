<objective>
Implement a mock ADCP projector server that simulates the real projector's protocol for development and testing. Supports both SHA256 auth and NOKEY modes, tracks power state, processes known commands, and can run standalone or as a pytest fixture. After this plan, developers can test the full ADCP pipeline without projector hardware.
</objective>

| Hash comparison timing attack | LOW | V2.9 | Use `hmac.compare_digest()` for hash comparison in mock server |

**Residual risk:** None significant — mock server is a development tool, not production.

Create `src/projector_bridge/mock_server.py` with:

```python
class MockProjector:
    def __init__(self, host="127.0.0.1", port=53595, password="Projector", nokey=False):
```

**State tracking:**
- `self.power_state: str = "standby"` (values: "standby", "on")
- `self.input_state: str = "hdmi1"` (values: "hdmi1", "hdmi2")
- `self.commands_received: list[str] = []` (for test assertions)
- `self.server: asyncio.Server | None = None`

**Methods:**

`async def start(self) -> None:` — Start the TCP server with `asyncio.start_server(self._handle_client, self.host, self.port)`

`async def stop(self) -> None:` — Close the server and wait for it to finish

`def get_port(self) -> int:` — Return the actual port (useful when started with port=0)

`async def _handle_client(self, reader, writer) -> None:` — Handle one client connection:
1. If `self.nokey`: send `NOKEY\r\n`
2. Else: generate challenge with `secrets.token_hex(8)`, send `{challenge}\r\n`, read auth hash, validate with `hmac.compare_digest(received_hash, expected_hash)`, send `ok\r\n` or `err_auth\r\n`
3. Read command line
4. Append raw command to `self.commands_received`
5. Call `self._process_command(command)` to get response
6. Send `{response}\r\n`
7. Close writer

`def _process_command(self, command: str) -> str:` — Process a command and return response string:
- `power "on"` -> set power_state="on", return "ok"
- `power "off"` -> set power_state="standby", return "ok"
- `power_status ?` -> return `"{self.power_state}"` (quoted)
- `input "hdmi1"` / `input "hdmi2"` -> set input_state, return "ok"
- `key "..."` -> return "ok" (any key command accepted)
- `blank "on"` / `blank "off"` -> return "ok"
- Unknown command prefix -> return "err_cmd"
- Known command with bad value -> return "err_val"

**Standalone entry point:**

```python
async def run_standalone(host="127.0.0.1", port=53595, password="Projector", nokey=False):
    server = MockProjector(host=host, port=port, password=password, nokey=nokey)
    await server.start()
    print(f"Mock projector listening on {host}:{server.get_port()}")
    print(f"Auth mode: {'NOKEY' if nokey else 'SHA256'}")
    try:
        await asyncio.Event().wait()  # run forever
    finally:
        await server.stop()
```

Add `if __name__ == "__main__":` block with argparse for `--host`, `--port`, `--password`, `--nokey` flags.
