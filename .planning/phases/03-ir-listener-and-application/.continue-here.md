<decisions_made>

- KY-022 (VS1838B) replaces TSOP38238 — available hardware, functionally equivalent for 38kHz IR
- Dispatch on EV_MSC/MSC_SCAN directly — kernel doesn't emit EV_KEY without a loaded keymap
- power_toggle queries power_status then sends appropriate on/off — remote has single button, ADCP needs explicit state
- ADCP commands sourced from Bcukier/sony_projector_adcp GitHub repo (power, input, picture_mode, brightness, contrast, sharpness, real_cre, key navigation)
- Wired KY-022: Signal->GPIO18 (Pin 12), VCC->3.3V (Pin 1), GND->Pin 6
</decisions_made>
